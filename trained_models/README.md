# trained_models

This directory stores local training and inference artifacts.

## Tracked files

- `ensemble_weights.txt`: model list used for ensemble inference and submission packaging.
- `README.md`: this file.

## Ignored files

- `*.pth`: checkpoint binaries generated by training.
- `tune_*/`: threshold tuning outputs generated by `tune_threshold.py`.

## Ensemble file format

`ensemble_weights.txt` expects one relative checkpoint path per line.

Example:

```text
trained_models/A_best.pth
trained_models/C_best.pth
```

Lines starting with `#` and empty lines are ignored.
